------------------------------------------------
QUIXEL COLORS v1.2 BETA
------------------------------------------------

Quixel Colors is a tool for Maya & Max users to assist in assigning Material IDs.

Maya:
------------------------------------------------
1 - Go to Windows > General Editors > Script Editor
2 - Select the Python taab
3 - Click File > Load Script
4 - Navigate to the colors directory inside SUITE 2.0s
5 - Open MayaRun.py
6 - Make sure that the directory on line 5 is correct
    If you have installed SUITE 2.0 to it's default directory, this should be fine.
7 - Press CTRL + Enter, or hit the blue single play button.

Max (Requires 3Ds Max 2014 Extension 1 or Higher):
------------------------------------------------
1 - Go to Utilities tab (hammer) on the top right of the command panel.
2 - Select MaxScript, and Run Script
3 - Navigate to color's directory 
4 - Run MaxRun.py
